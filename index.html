Lesson 1 Notes:-

Basics:-

We use a language called HTML to write web pages. HTML stands for Hyper Text Markup Language.
We use a protocol called HTTP to transfer web pages to our browser. HTTP stands for Hyper Text Transfer Protocol
The WWW (World Wide Web) was invented in 1990 and has around 30 Billion pages

Some HTML we learned about:-

Anchors:- Used to insert links into web pages
Void Tags:- A tag with no closing tag such as "br"
Inline tags:- Tags that are displayed "inline" on the web page, such as "em" and "span"
Block tags:- Tags that have a shape that forms a box around the content, such as "p" and "div"

Lesson 3 Notes:-

CSS is used to control the style of HTML elements. CSS is based on inheritance. This allows us to use CSS to reduce code used when declaring properties of elements. We try and avoid repetition because it saves on code when we reuse, and it's easier to debug. The CSS developer guide is located here. CSS format is selector:property:value.

We can include CSS in 3 ways:-

  • In the head of the HTML
  • Link to a separate CSS file
  • Write style inline with the HTML

In CSS code comments begin with /* and end with */

Each HTML element is a box, and each box has 4 components. We can set the size of the box using pixels, percentage or the box-sizing attribute to border-box

When writing CSS use the theory - Code, Test, Refine. 1) Look for natural boxes. 2) Look for repeated styles. 3) Write the HTML. 4) Apply styles from biggest to smallest. 5) Fix things.

To verify HTML we use HTML Verify

To verify CSS we use CSS Verify
